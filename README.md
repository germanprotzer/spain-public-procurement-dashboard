\# ⭐ Spain Public Procurement Dashboard



This repository contains a Power BI portfolio project focused on public procurement opportunities in Spain.



The dashboard is designed to help companies identify where public spending is concentrated, which contracting authorities are publishing the largest tenders, which regions show the highest pipeline value, and where short-term opportunities may exist.



\---



\## Project overview



This dashboard analyzes Spanish public procurement data and transforms it into a more business-friendly view through two main pages:



\- \*\*Pipeline Overview\*\*  

&#x20; High-level view of active procurement opportunities, including forecast, weighted pipeline, open pipeline, top contracting authorities, procurement stages, and top regions.



\- \*\*Opportunity Explorer\*\*  

&#x20; Interactive analysis of procurement opportunities by sector, region, procedure, and contracting authority.



\---



\## Key business questions



The dashboard helps answer questions such as:



\- Where is government spending concentrated?

\- Which sectors have the strongest procurement pipeline?

\- Which public institutions publish the biggest opportunities?

\- Which regions show the highest procurement activity?

\- What short-term opportunities are available for companies?
